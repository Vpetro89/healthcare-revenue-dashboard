## Healthcare Revenue Cycle Tableau Dashboard

### Description
This project contains a Tableau dashboard built to review healthcare revenue cycle performance. It summarizes billed, allowed, and paid revenue, shows billing trends over time, and compares paid revenue across clinical departments. The data started in Excel and was structured for Tableau to reflect the kind of operational revenue reporting commonly used in healthcare settings.

### Dashboard Focus
- Total billed revenue
- Total allowed revenue
- Total paid revenue
- Monthly billing trends
- Paid revenue by clinical department

### Data Source
- Excel dataset prepared for Tableau analysis

### Tools
- Tableau
- Excel

### Output
- Tableau dashboard for healthcare revenue reporting
